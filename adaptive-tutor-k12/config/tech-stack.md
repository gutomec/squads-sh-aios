# Tech Stack — adaptive-tutor-k12

## Plataformas de Aprendizagem
- **Khan Academy** — Conteúdo educacional gratuito com exercícios adaptativos
- **IXL** — Plataforma de prática personalizada K-12 com diagnósticos
- **Duolingo** — Aprendizagem gamificada com spaced repetition
- **Google Classroom** — Gestão de sala de aula e distribuição de conteúdo

## Padrões Curriculares
- **BNCC** (Base Nacional Comum Curricular) — Padrão curricular brasileiro K-12
- **Common Core** (Common Core State Standards) — Padrão curricular dos EUA (ELA e Matemática)
- **National Curriculum** (UK) — Currículo nacional britânico (Key Stages 1-4)

## Adaptive Learning
- **Knewton** — Motor de aprendizagem adaptativa baseado em dados
- **DreamBox** — Matemática adaptativa para K-8
- **ALEKS** — Assessment and Learning in Knowledge Spaces (avaliação adaptativa)

## Assessment
- **Bloom's Taxonomy** — Taxonomia de objetivos educacionais (6 níveis cognitivos)
- **Webb's DOK** — Depth of Knowledge (4 níveis de profundidade)
- **Formative Assessment Tools** — Ferramentas de avaliação formativa contínua

## Analytics
- **Learning Analytics** — Análise de dados de aprendizagem para otimização
- **xAPI** (Experience API) — Padrão para rastreamento de experiências de aprendizagem
- **SCORM** — Sharable Content Object Reference Model

## Comunicação
- **Email** — Relatórios periódicos para pais e educadores
- **WhatsApp** — Comunicação direta com pais (mercado brasileiro)
- **Portal de Pais** — Dashboard online para acompanhamento de progresso
