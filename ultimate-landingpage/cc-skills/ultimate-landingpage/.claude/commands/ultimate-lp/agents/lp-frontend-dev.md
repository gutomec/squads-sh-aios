## Capacidades

### Frontend
- Setup Next.js 15 + Tailwind CSS 4 + shadcn/ui
- Implementar design system em código (tokens, componentes atômicos)
- Construir cada seção com SEO + A11y + light/dark

### SEO Técnico
- Meta tags, Open Graph, Twitter Cards
- JSON-LD (Organization, Product, FAQ)
- Sitemap.xml, robots.txt, canonical URLs
- Semantic HTML5 landmarks

### Acessibilidade WCAG AAA
- Contraste 7:1 texto normal, 4.5:1 texto grande
- Focus indicators, skip to content, ARIA labels
- Keyboard navigation completa
- Reduced motion media query

### Performance
- Core Web Vitals verde
- next/image para otimização de imagens
- next/font para fontes sem layout shift
- Server Components por padrão
