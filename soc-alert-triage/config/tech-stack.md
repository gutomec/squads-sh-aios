# Tech Stack — soc-alert-triage

## SIEM (Security Information & Event Management)
- **Splunk** — Enterprise SIEM, log analytics e correlação de eventos
- **IBM QRadar** — SIEM e análise de ameaças com AI
- **Microsoft Sentinel** — Cloud-native SIEM integrado ao Azure
- **Elastic SIEM** — Detecção e resposta baseada no Elastic Stack

## IDS/IPS (Intrusion Detection/Prevention Systems)
- **Snort** — IDS/IPS open-source com detecção por assinatura
- **Suricata** — IDS/IPS de alto desempenho multi-threaded
- **Palo Alto Networks** — Next-gen firewall com IPS integrado

## EDR (Endpoint Detection & Response)
- **CrowdStrike Falcon** — EDR e threat hunting cloud-native
- **SentinelOne** — EDR autônomo com AI
- **Carbon Black** (VMware) — EDR e prevenção comportamental
- **Microsoft Defender for Endpoint** — EDR integrado ao ecossistema Microsoft

## Threat Intelligence
- **MISP** — Malware Information Sharing Platform (open-source)
- **VirusTotal** — Análise de malware, IOC lookup e reputation
- **AbuseIPDB** — Reputation database para IPs maliciosos
- **AlienVault OTX** — Open Threat Exchange (feeds gratuitos)
- **Recorded Future** — Threat intelligence com AI/ML

## SOAR (Security Orchestration, Automation & Response)
- **Splunk Phantom** — Automação de resposta a incidentes
- **Palo Alto XSOAR** (Cortex) — Orquestração e automação SOC
- **Swimlane** — SOAR com low-code automation

## Frameworks de Referência
- **MITRE ATT&CK** — Framework de táticas, técnicas e procedimentos (TTPs)
- **NIST CSF** — Cybersecurity Framework (Identify, Protect, Detect, Respond, Recover)
- **OWASP** — Open Web Application Security Project

## Email Security
- **Proofpoint** — Proteção contra phishing e BEC
- **Mimecast** — Segurança de email e continuidade
- **Microsoft Defender for Office 365** — Proteção de email cloud

## Cloud Security
- **AWS GuardDuty** — Detecção de ameaças para AWS
- **Azure Defender** — Proteção de workloads cloud
- **GCP Security Command Center** — Posture management e detecção
